# PalindromeCheckerApp

## Running Procedure
---

Run the program `PalindromeCheckerApp.java` by first compiling the file and then clicking the **Run** button in IntelliJ IDEA.

## Flow of Project
---

1. Program starts.
2. JVM invokes the `main()` method.
3. Application name is displayed.
4. Application version is displayed.
5. Program continues to the next use case or exits.

## Objective
---

To implement a **Palindrome Checker** that determines whether a given input is a palindrome using an efficient solution.
